--- layout: post title: "Hola mundo ASP.NET" date: 2014-02-05 20:37:00 categories: coding --- El ejemplo minimalista de una pagina asp.net con su extension aspx
Simplemente tiene un control label y muestra el mensaje standar de iniciacion 
<!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.0 Transitional//EN">
<HTML>

	<!-- Este script se ejecutara en el servidor -->
	<script language="VB" runat="server">
		Public Sub Page_Load()
			lblSaludo.Text = "Hola, nena"
		End Sub
	</script>
	<HEAD>
		<title>Hola nena</title>
	</HEAD>
	<body>
		<form id="Formulario" runat="server">
			<!-- Esto es un control de servidor -->
			<asp:Label id="lblSaludo" runat="server" />
		</form>
	</body>
</HTML>

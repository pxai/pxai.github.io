--- layout: post title: "Funciones y librerias ASP" date: 2001-02-05 16:35:00 categories: asp3, web --- Un par de ejemplos supersimples de funciones y el uso de librerias.
Ejemplo de ASP con funciones simples: <!-- ejemplo de libreria --> <!-- procedimiento para mostrar cabecera --> <% Sub cabecera(titulo) %> <html> <head> <meta http-equiv="Content-Type" content="text/html; charset=windows-1252"> <meta name="GENERATOR" content="Microsoft FrontPage 4.0"> <meta name="ProgId" content="FrontPage.Editor.Document"> <title>Pagina ASP <%=titulo %></title> </head> <% end Sub %> <% ' una funcion que invoca a trim y devuelve el resultado function limpia(cadena) limpia = trim(cadena) end function ' function que invoca a replace function sustituye(cadena, orig,cambio) sustituye= replace(cadena,orig, cambio) end function %> Y aqui, se invoca la libreria con una especie de comentario HTML <!-- ejemplo de invocacion de libreria --> <!-- Inclusion de libreria --> <!-- #include file="asp06.asp" --> <!-- invocamos el procedimiento para mostrar cabecera --> <% cabecera("07 - utilizacion de librerias") %> <body> libreria usada: asp06.asp </body> </html>
